# Frontends Projects - Password Generator 
___

## Обзор
Функционал:
 - Генерация случайного пароля
 - Возможность выбрать:
 -- Длину пароля(от 4 до 20)
 -- Верхний регистр букв
 -- Нижний регистр букв
 -- Числа
 -- Символы
___

## Использованные технологии
* HTML
* CSS
* JS
* Иконочный стиль (Font Awesome)
